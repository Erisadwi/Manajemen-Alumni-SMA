# 🎓 Manajemen Alumni SMA

Sistem Informasi Manajemen Alumni SMA berbasis web yang dibangun menggunakan PHP dan MySQL. Aplikasi ini bertujuan untuk membantu sekolah dalam mengelola data alumni secara terstruktur, memudahkan pencarian data alumni, serta menyediakan media pengelolaan informasi alumni secara digital.

## 📌 Fitur Utama

* Login dan Logout
* Dashboard Admin
* Manajemen Data Alumni
* Tambah Data Alumni
* Edit Data Alumni
* Hapus Data Alumni
* Pencarian Data Alumni
* Penyimpanan Data ke Database MySQL
* Tampilan berbasis web yang mudah digunakan

## 🛠️ Teknologi yang Digunakan

* PHP
* MySQL
* HTML
* CSS
* Bootstrap
* JavaScript
* XAMPP

## 📋 Modul Sistem

### Admin

* Login
* Logout
* Kelola Data Alumni
* Tambah Alumni
* Edit Alumni
* Hapus Alumni

### Alumni

* Melihat informasi data alumni
* Pencarian data alumni

## 🎯 Tujuan Sistem

Sistem ini dibuat untuk membantu pihak sekolah dalam:

* Mengelola data alumni secara terpusat.
* Mempermudah pencarian informasi alumni.
* Mengurangi proses pendataan secara manual.
* Meningkatkan efisiensi pengelolaan data alumni.
